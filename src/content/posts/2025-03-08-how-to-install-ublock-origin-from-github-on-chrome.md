# How to Install uBlock Origin from GitHub on Chrome

uBlock Origin is one of the most popular and powerful ad-blockers available. While you can easily install it from the Chrome Web Store, some users prefer to install it manually from GitHub—especially for testing the latest updates or avoiding store restrictions. Here’s how you can do it step by step.

---

## **Step 1: Download the Latest Release**

To get the latest version of uBlock Origin directly from GitHub:

1. Open your browser and visit the official uBlock Origin GitHub repository:  
   👉 [https://github.com/gorhill/uBlock](https://github.com/gorhill/uBlock)
2. Click on the **“Releases”** section on the right-hand sidebar.
3. Scroll down to find the latest release and look for the **uBlock0.chromium.zip** file.
4. Download the **ZIP file** to your computer.

---

## **Step 2: Extract the Files**

Once the ZIP file is downloaded:

1. Locate the downloaded file on your computer.
2. Right-click and select **“Extract All”** (Windows) or **“Extract Here”** (Mac/Linux).
3. Note the folder where the files are extracted.

---

## **Step 3: Enable Developer Mode in Chrome**

To manually install the extension, you need to enable Developer Mode:

1. Open **Google Chrome**.
2. In the address bar, type:  
   👉 `chrome://extensions/`  
   and press **Enter**.
3. Toggle the **“Developer mode”** switch in the top-right corner.

---

## **Step 4: Load the Extension**

Now, you can install uBlock Origin manually:

1. Click the **“Load unpacked”** button (top-left corner of the Extensions page).
2. Navigate to the folder where you extracted the uBlock Origin files.
3. Select the **“uBlock0.chromium”** folder and click **Open**.
4. Chrome will install the extension, and the **uBlock Origin icon** (a red shield with a white “μ”) should appear in your toolbar.

---

## **Final Thoughts**

Congratulations! 🎉 You have successfully installed uBlock Origin from GitHub. This method ensures that you get the latest updates or beta versions without waiting for the Chrome Web Store to approve them.

### **Need Help?**
If you run into any issues, check out the [official uBlock Origin wiki](https://github.com/gorhill/uBlock/wiki) or leave a comment below. Enjoy ad-free browsing! 🚀
