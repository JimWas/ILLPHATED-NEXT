🔧 Troubleshooting Tips
Addon Not Showing Up: Ensure that the folder name matches the .toc file name and that the folder is placed directly inside the AddOns directory.

Nested Folders: Some addons may extract with an extra top-level folder (e.g., AddonName-master/AddonName/). Ensure that the folder containing the .toc file is directly inside the AddOns directory.
Turtle WoW

Memory Settings: If you're using multiple addons, set the addon memory usage to 0 (no limit) in the AddOns menu at the character selection screen.
Turtle WoW

Compatibility: Turtle WoW uses the 1.12 client. Addons designed for later versions (e.g., Classic) may not be compatible.
